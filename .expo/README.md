# Daily Diet

O **Daily Diet** é um projeto desenvolvido para o acompanhamento de refeições diárias, permitindo ao usuário manter o controle de sua dieta de forma simples e visual. O projeto foi construído utilizando **React Native** e **TypeScript**.

## Funcionalidades

- **CRUD de Refeições**: Cadastre, edite, visualize e remova refeições.
- **Estatísticas Dinâmicas**: Cálculo automático de melhor sequência (streak), total de refeições e porcentagem dentro da dieta.
- **Feedback Visual**: Cores dinâmicas baseadas no status da dieta (Verde/Vermelho).
- **Persistência Local**: Armazenamento seguro utilizando AsyncStorage.
- **Multiplataforma**: Compatibilidade ajustada para dispositivos móveis e Web.

## Tecnologias Utilizadas

- [React Native](https://reactnative.dev/)
- [TypeScript](https://www.typescriptlang.org/)
- [Styled Components](https://styled-components.com/)
- [React Navigation](https://reactnavigation.org/)
- [Phosphor Icons](https://phosphoricons.com/)
